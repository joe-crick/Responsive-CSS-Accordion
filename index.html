Pure Extended

CSS Only Components for Pure CSS.

Pure CSS

Pure CSS is a great set of small, responsive CSS modules. As their docs say, it is tiny. The entire set of modules is only 3.8KB* (minified and gzipped).

Pure is responsive out of the box.

Pure has minimal styles. It encourages you to write your application styles on top of it. Pure does not look like this page out of the box. The styles for this page were blatantly and lovingly copied from Pure CSS' home page. Pure is designed to get out of your way. It is easy to override styles. You can use it alongside Bootstrap, if you want (as you can Pure-extended).

For details on Pure, see their website (Pure CSS)

Pure doesn't have any components. Boo. So, I wrote some. My goal is to get parity with Bootstraps menu of components. I'm very close.

Installation

Pure-extended offers two builds:

  • Bundled: This build has Pure CSS bundled with Pure-extended
  • Stand Alone: Just Pure-extended

You can install Pure-extended from npm, download it, or get it from a CDN 

                            yarn add pure-extended
