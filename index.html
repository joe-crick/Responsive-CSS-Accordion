Pure Components

CSS Only Components for Pure CSS.

Pure CSS

Pure CSS is a great set of small, responsive CSS modules. As their docs say, it is tiny. The entire set of modules is only 3.8KB* (minified and gzipped).

Pure is responsive out of the box.

Pure has minimal styles. It encourages you to write your application styles on top of it. Pure does not look like this page out of the box. The styles for this page were blatantly and lovingly copied from Pure CSS' home page. Pure is designed to get out of your way. It is easy to override styles. You can use it alongside Bootstrap, if you want (as you can with pure-components).

For details on Pure, see their website (Pure CSS)

Pure doesn't have any components, so I wrote some. My goal is to get parity with Bootstrap's menu of components. I'm very close. Please note, this is still a work in progress. The usual cations apply.

Installation

Pure-components offers two builds:

  • Bundled: This build has Pure CSS bundled with Pure-components
  • Stand Alone: Just Pure-components

You can install Pure-components from npm, download it, or get it from a CDN. 

<link
  rel="stylesheet"
  href="https://unpkg.com/purecss-components@0.0.7/dist/pure-components.css"
  integrity="sha384-FfmMHw1nM6QpEpUtacs5k0pNzo88/JlQAevyYjewWTklEKxFsFyH2nbVdbX6xCf4"
  crossorigin="anonymous">

yarn add pure-components

or

npm i pure-components
